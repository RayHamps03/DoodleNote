# DoodleNote
Doodle Note is a social website in development where users have drawing boards to create doodles of anything they can imagine to share as notes to other users on the site. 

## Upcoming Features
There are many upcoming features that are planned to be implemented into the website, including:
- Functional login system
- Drawing board to create doodles
- Dashboard home page to show notes as uploaded doodles
- Like and comment system to interact with other user's doodles
- Follower system for users to follow other users to keep up to date on recent notes

## Technologies/Frameworks used
- Entity Framework Core
- ASP.NET Core MVC with C#
- Bootstrap 5
- HTML/CSS/JS
